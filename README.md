# MoneyTransferRESTApi - A dead simple Java REST API(without Spring) <br> to transfer money between accounts <br>

### Quick Start - Run
1.<br>
```$xslt
./gradlew build
```
2.<br>
```$xslt
./gradlew startApp (App will be running on http://localhost:8080)
```
### Run All Tests
```$xslt
./gradlew test
```
